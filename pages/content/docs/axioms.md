# 公理系

以下の 11 項で閉じている。**これ以上足さない方針。**

1. **tick**：1ヶ月単位。1年 = 12ヶ月。全 human が毎月同時手番でアクションを決定する（lockstep）。
2. **寿命**：human は死ぬ。
3. **能力曲線**：身体能力（stats）は年齢の関数として変化し、実行可能なアクションを決める。
4. **環境循環**：world は human のほかに**環境ストック**を持ち、組成は human と環境の間を閉じて循環する。harvest（skill の発動 → [03-skills.md](./skills.md)）は環境から resource を取り出し、**獲得量は環境ストックの残量に依存する**（枯渇）。消費は resource を消滅させず**廃棄物 resource に変換**し、環境への還元は**廃棄（discard）と死亡時還元によってのみ**起きる。環境は時間経過で resource を別の resource に変換し（光合成・分解のイメージ）、**g を増やせるのはこの環境変換だけ**。増分は外部からの自由エネルギー流入 Φ が上限を与える。（[world.md](./world.md)）
5. **組成保存**：resource は隠れた組成ベクトルを持ち、全変換で組成を厳密に保存する。
6. **ε（偶発的出会い）**：知人獲得は紹介（triadic closure）が基本で、確率 ε でランダムな出会いが起きる。ε が world の唯一の地理パラメータ。
7. **brain 継承**：0〜6歳は world 提供の共通 baby brain。6歳時点で父母どちらかの brain を 50% ずつの確率で引き継ぐ。
8. **識別不能**：brain の同一性は観測できない。観測できるのは行動の履歴のみ。
9. **Shapley 採点**：brain のスコアは、ランダムな brain の組み合わせで多数の world を回したときの、世界の総価値 v(S) への限界寄与の期待値。
10. **親密度**：human のペアは親密度を持つ。値は当事者二人だけに可視で、両者に同じ値が見える。毎月の増減が両者の stats から生じ、当事者間のアクションのやり取りでも増減する。（[human.md](./human.md)）
11. **空間**：world の空間は有限の数値量である。human は身体と保有 resource の体積の分だけ空間を占有する。空間は生成も破壊もされず、ゼロ次元世界で human が取り合う唯一の「場所」である。（[world.md](./world.md)）

## 三つの保存クラス

| クラス | 保存 | 移転 | 備考 |
| --- | --- | --- | --- |
| **resource** | 保存量 | 可 | 変換（invoke → [03-skills.md](./skills.md)）で形を変えるが組成は保存される |
| **state（stats）** | 非保存 | 不可 | 訓練で上げ、老化で下がる |
| **skill** | 非保存 | 非競合 | 教えても減らず、コピーで増える |

skill は**世界で総量が増えうる唯一の量**であり、経済成長の源泉はここにしかない。

親密度（公理 10）は state クラスの対人版：非保存・移転不可で、ペアが対で共有する。
