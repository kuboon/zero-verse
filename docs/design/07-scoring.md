# 採点：Shapley 値

- v(S) への限界寄与のモンテカルロ推定。組み合わせサンプリングは毎ラウンド振り直す（固定相手へのオーバーフィット防止）。world サイズで正規化する。
- **v(S) = 各 human の消費 c に対する凹効用 u(c) の総和**。u の凹性により、同じ総生産でも平等な分配が v を大きくする。搾取戦略は限界寄与が負になり、定義から罰される。
- 消費 c の実体は、廃棄物モデル導入後は「stats 獲得の invoke で失われた自由エネルギー **Δg = g(入力) − g(産出物)**」と測るのが正確（resource 自体は消滅しないため。[02-resources.md](./02-resources.md)）。食事なら「食べ物の g − 廃棄物の g」。厳密な定義は Shapley 実装時に確定するが、M1 の生涯消費の計測もこの定義を仮採用する（[PLAN.md](../PLAN.md)）。
- 凹効用により、多産か少数精鋭（手厚い養育）かが brain の選択になり、人口転換が創発しうる。
- Shapley 採点は同一戦略の複製に限界寄与の希釈で罰を与える。メタが単一支配戦略に収束せず、ニッチ（教師特化、仲買特化、探索特化）の生態系になる。
- **未決**：v の時間扱い。生存線超過消費の年数総和にするか、割引率を入れるか。割引率を world パラメータ化する案あり（[90-open-questions.md](./90-open-questions.md) 参照）。
