# 経緯メモ（破棄した案）

同じ案を再提案しないための記録。

| 破棄した案 | 破棄理由 | 代替 |
| --- | --- | --- |
| 人口をスコアにする | マルサス的に一人あたり豊かさが生存線に張り付く | 凹効用の消費総和（[07-scoring.md](./07-scoring.md)） |
| 婚姻を子の帰属交渉契約にする | 合理的 brain は子を売らず市場が死ぬ | 50/50 継承（[05-kinship.md](./05-kinship.md)） |
| クラスタごとに resource の呼称が分岐する | brain は名前を読まないため効かない。分岐するのは名前でなく法則の仮説であり、接触時に起きるのは翻訳でなく検証 | 呼称の分岐は人間参加モード専用の楽しみとして残す |
| 1 world = 1 Durable Object で全処理する | 大規模ランは CPU 制約で不可 | ネイティブ実行と DO の二層（[08-architecture.md](./08-architecture.md)） |
